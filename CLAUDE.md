# Syspeek - Claude Code Instructions

## Build

- **Assets embebidos** - Los archivos estáticos (CSS, JS, HTML, íconos) están embebidos en el binario. Después de modificar cualquier archivo en `static/` o `templates/`, es necesario recompilar con `go build` para que los cambios surtan efecto.

## Git Commits

- **No atribuir commits a Claude** - No usar la firma "Generated with Claude Code" ni "Co-Authored-By: Claude" en los commits de este proyecto.
